This repository contains all the example codes and exercises included in the **Do It Golang!** book.  
Each chapter is organized into its own directory and includes various examples ranging from **basic Go syntax** to **advanced topics**.

---

### üìÇ Directory Structure

| Chapter/Folder | Description |
|----------------|-------------|
| `ch01` | Introduction to Go, installation, and first Go program examples |
| `ch02` | Go development environment setup, environment variables, external packages |
| `ch03` | Examples on variables, constants, data types, and local/global variables |
| `ch04` | Examples of basic data types, type conversion, and pointer usage |
| `ch05` | Examples of arithmetic, comparison, logical, bitwise, and other operators |
| `ch06` | Examples of function declarations, multiple return values, anonymous functions, and closures |
| `ch07` | Examples of conditional statements (`if`, `switch`) and logical operators |
| `ch08` | Examples of loops (`for`), loop control, and multiplication table program |
| `ch09` | Examples of data structures such as arrays, slices, maps, and structs |
| `ch10` | Examples of OOP concepts, methods, interfaces, receivers, and inheritance |
| `ch11` | Examples of standard I/O and file I/O |
| `ch12` | Examples of error handling, `panic` and recovery, and using `defer` |
| `ch13` | Examples of concurrency programming, goroutines, and newsletter fetching |
| `ch14` | Examples of concurrency control with channels, select statements, locks, and wait groups |
| `ch15` | Examples of generics with functions, types, and interfaces |
| `ch16` | Examples of performance optimization, profiling, and profile-based optimization |
| `ch17` | Examples of networking concepts (TCP, UDP, HTTP) |
| `ch18` | Implementation of a task manager and book info management web application |
| `ch19` | Examples of unit testing, assertions, test coverage, mocking, and stubbing |
| `ch20` | Examples of refactoring concepts, code improvement, and structure redesign |
| `appendix` | Appendix: Examples using `godoc`, `context`, and `cgo` packages |
| `extra` | Additional examples such as higher-order functions |
